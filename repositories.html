Source Repositories
Github Master Repositories

The Master repositories can be cloned by doing the following

git clone git://github.com/jfm/RateBeer-Mobile.git

Testing Project Repositories

The repositories for the testing project can be cloned by doing the following

git clone git://github.com/jfm/RateBeer-Mobile-Test.git

Contributing to the Project

If you want to start contributing code to the project, you should do the following:

  • Create a profile on github.com.
  • Go to: http://github.com/jfm/RateBeer-Mobile
  • Click the Fork button to create your own private fork of the project.
  • The do the following:

    git clone

  • To be able to get updates from the master repository do the following:

    git remote add rbmaster git://github.com/jfm/RateBeer-Mobile.git

  • To get updates from the master branch in the master repository you would then do:

    git pull rbmaster master
